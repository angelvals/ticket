Diferencias entre principales identificadores

Los principales identificadores usados son id y class pero puedes alterar las propiedas de cada elemento usando solo el nombre del elemento, estos son los principales identificadores que usaremos:

  • Class: se usa . para seleccionar el elemento
  • ID: se usa # para seleccionar el elemento
  • Nombre: este se ultimo es llamado usando el nombre de la etiqueta que quieras seleccionar, por ejemplo si la etiqueta se llama img puedes usar img en css para seleccionar el elemento
